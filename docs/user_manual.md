# Cognify User Manual

## Overview
Cognify is an integrated study and productivity application designed to reduce friction between capturing information and converting it into study artifacts.

## Features
1. **Notepad**: A lightweight text editor for creating and organizing notes.
2. **Concept Map**: A visual minimap that links ideas and notes to show relationships.
3. **Apollo AI Assistant**: A context-aware study coach that suggests summaries, generates flashcards, and links notes to calendar events.
4. **Flash Cards**: Digital cards for spaced repetition study.
5. **Calendar Integration**: Create and sync calendar events with your study sessions.

## Quick Start
1. **Register/Login**: Use your email and password to access the app.
2. **Dashboard**: View your schedule, upcoming flashcards, and recent notes.
3. **Notes**: Create a new note and use the AI button to generate summaries and flashcards automatically.
4. **Flashcards**: Navigate to the flashcard section to review cards scheduled for today based on spaced repetition.
